== Description ==

Create a talk style design with a visual editor Gutenberg.
Gutenberg Editor Blocks Plugin. Compatible with AMP.

= Settings =

* Avatar, Name
* Direction: Left, Right
* Design: Default, Bubble, Square, Dashed, Shadow, Borderless
* Size: Default, Small, Large
* Options: Default, Short, Vertical
* Background color, Text color
* latest information on [LIQUID PRESS](https://lqd.jp/wp/).

== Installation ==

1. Upload 'liquid-speech-balloon' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set 'LIQUID SPEECH BALLOON' in the 'Settings' menu
